# UI Components Tracker

File ini dibuat untuk memudahkan pencatatan dan pengecekan komponen UI (dari folder `app/ui`) di dalam proyek ini.
Anda bisa menggunakan tanda `[x]` untuk menandai status, dan `[ ]` jika belum/tidak ada.

**Keterangan Kolom:**
- **Terpakai:** Apakah komponen ini sedang di-*import* dan digunakan?
- **Error:** Apakah komponen ini sedang memiliki *bug* atau *error*?
- **Proses Update:** Apakah komponen ini sedang dalam tahap perombakan/penyesuaian?
